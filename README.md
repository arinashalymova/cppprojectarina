# cppprojectarina

Проект представляет собой приложение, с возможностью просмотра функционала на  http://localhost:8080, и не подходит для установки на GitHub Pages, тк является сервесрным скриптом, а не статическим веб-сайтом. К сожалению невозможно установить на ноутбук с которого ведется разработка необходимые архивы((

Идея реализации: Шалымов Арина, Старцев Захар.


Бекенд и основной функционал, работа с txt для сохранения данных: Шалымова Арина


Привязка хоста, фронтенд, доработка методов (фишки): Старцев Захар


Тестирование, исправление багов: Шалымова Арина, Старцев Захар
